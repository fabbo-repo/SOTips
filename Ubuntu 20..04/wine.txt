Instalar Wine:
- Habilitar la arquitectura de 32 bits ejecutando la siguiente línea en la terminal
-> sudo dpkg --add-architecture i386
- Descargar repositorio de Wine
-> sudo wget -nc https://dl.winehq.org/wine-builds/winehq.key
- Agregar la clave de Wine
-> sudo apt-key add winehq.key
- Añadir los repositorios de Wine
-> sudo apt-add-repository 'deb https://dl.winehq.org/wine-builds/ubuntu/ eoan main'
- Actualizar los paquetes:
-> sudo apt update
- Instalar Wine
-> sudo apt install --install-recommends winehq-stable
- Comprobar que está instalado
-> wine -–version
- Entrar en la configuración de Wine
-> winecfg
- Dar en instalar para que instale wine-mono (si se requiere intalar los paquetes adicionales)
-> Configurar Wine
